
# Мини-оболочка с файловыми командами на Python

Простая командная оболочка на Python с набором файловых операций.

### Установка
Клонируйте репозиторий
git clone <[ссылка_на_репозиторий](https://github.com/jurolli/Lab_2/tree/main/src)>


### Используйте pip
pip install -r requirements.txt## Запуск
h
cd src
python main.py## Команды


### Базовые операции

- `ls [path] [-l]` - просмотр содержимого директории
- `cd [path]` - смена директории (поддерживает `..`, `~`)
- `cat [file]` - просмотр содержимого файла
- `cp [source] [destination] [-r]` - копирование файлов/директорий
- `mv [source] [destination]` - перемещение/переименование
- `rm [file] [-r]` - удаление файлов/директорий
